# Judo Flashcards

A simple flashcard app for studying judo terminology and techniques.

## Features

- Study by category (Vocab or Perform)
- Tracks correct/wrong answers
- Prioritizes cards you've gotten wrong
- Progress saved in browser
- Works on mobile and desktop

## Usage

1. Select a category from the dropdown
2. Click "Start Practice"
3. Click the card to flip and see the answer
4. Mark as Correct or Wrong

## Keyboard Shortcuts

- `Space` / `Enter` - Flip card
- `Arrow keys` - Previous/Next card
- `1` or `C` - Mark correct
- `2` or `W` - Mark wrong

## Adding Cards

Cards are stored as CSV files in `study/[belt]/vocab.csv` and `perform.csv`:

```csv
Question,Answer
"What is Uchi Mata?","Inner thigh throw"
```
